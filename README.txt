
Kasir Bu Rusna — PWA Bundle
===========================
- Proteksi sandi saat membuka tab **Laporan** (atur di Pengaturan → Keamanan Laporan).
- Tombol stok bisa diklik untuk edit (tanpa submit form).
- Kamera bekerja di HTTPS / http://localhost.
- Branding dan nama install: "Kasir Bu Rusna".

Deploy ke GitHub Pages:
1) Upload SEMUA isi folder ini ke repo (root).
2) Settings → Pages → Source: Deploy from a branch (main / root).
3) Buka URL https://<username>.github.io/<repo>/

Jika update tak muncul, naikkan versi di `sw.js` (mis. v2), lalu refresh.
